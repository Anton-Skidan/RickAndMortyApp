* Rick and Morty Characters App

Мобильное приложение на Flutter для просмотра персонажей мультсериала Rick and Morty с поддержкой избранного, офлайн-доступа, пагинации и кеширования изображений.

* Описание

Приложение загружает список персонажей с публичного API Rick and Morty, позволяет добавлять их в избранное и работать без подключения к интернету за счёт локального хранения данных в Hive.

* Функциональность

Отображение персонажей в виде карточек с основной информацией

Пагинация при достижении конца списка

Добавление и удаление персонажей из избранного

Экран избранного с возможностью сортировки

Офлайн-доступ к данным после первой загрузки

Кеширование изображений

Поддержка светлой и тёмной темы

Анимации при добавлении и удалении из избранного

* Используемые технологии

* Структура проекта

Проект построен по архитектуре BLoC и разделён на две основные фичи:

Экран со списком персонажей

Экран избранного

Отдельно вынесены:

Слой локального хранилища

Сетевой слой

Переиспользуемые UI-компоненты

Управление темой и конфигурацией приложения

Основной упор сделан на разделение ответственности и читаемость кода

* Установка и запуск

Flutter 3.19.0 или выше

Dart 3.3.0 или выше

* Запуск
  
git clone https://github.com/Anton-Skidan/RickAndMortyApp.git

cd rick_and_morty_app

flutter pub get

flutter run

* Работа офлайн

После первой загрузки данные сохраняются в локальное хранилище через Hive

При повторном запуске без подключения к интернету персонажи отображаются из кеша

* API

Используется публичный REST API Rick and Morty:

https://rickandmortyapi.com/
